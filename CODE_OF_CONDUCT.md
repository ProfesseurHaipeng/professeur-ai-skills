# Community conduct

This project welcomes careful technical disagreement and evidence-backed
critique. Participate with respect, describe behavior rather than attacking
people, and give contributors a reasonable chance to correct mistakes.

Unacceptable conduct includes harassment, threats, discriminatory language,
sexualized attention, doxxing, publishing private material, impersonation,
spam, manipulation of repository activity, and knowingly unsafe instructions.

Maintainers may edit or remove content, restrict participation, or report
serious abuse to the hosting provider. Enforcement decisions should be
proportionate, documented privately, and avoid redistributing sensitive
material.

Report conduct concerns through a private GitHub channel rather than a public
issue. Include only the minimum information needed for review.
