# Standalone Skill index

The six research Skills are intentionally not bundled into this repository. Install one repository at a time:

| Theme | Repository | Skill name | Release |
| --- | --- | --- | --- |
| I Ching | `ProfesseurHaipeng/iching-research-skill` | `iching-research` | `v0.1.0` |
| BaZi | `ProfesseurHaipeng/bazi-calendar-research-skill` | `bazi-calendar-research` | `v0.1.0` |
| Tarot | `ProfesseurHaipeng/tarot-reading-research-skill` | `tarot-reading-research` | `v0.1.0` |
| Western astrology | `ProfesseurHaipeng/western-astrology-research-skill` | `western-astrology-research` | `v0.1.0` |
| Shanhaijing | `ProfesseurHaipeng/shanhaijing-mythic-geography-skill` | `shanhaijing-mythic-geography` | `v0.1.0` |
| Plato | `ProfesseurHaipeng/plato-myth-symbolism-skill` | `plato-myth-symbolism` | `v0.1.0` |

Example:

```bash
gh skill install ProfesseurHaipeng/iching-research-skill \
  --skill iching-research \
  --pin v0.1.0
```

Use the independent repository's issue tracker for domain-specific improvements. Use this repository's Portable Skill Doctor to audit a Skill before installation or publication.
