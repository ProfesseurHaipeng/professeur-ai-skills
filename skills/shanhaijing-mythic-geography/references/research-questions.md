# Research questions

- How does chapter order create a route, catalogue, ritual perimeter, or cosmological diagram?
- When do the same names shift across editions or commentaries?
- Which creatures are tied to sound, diet, illness, omen, or sacrifice, and how do translators render those relations?
- What is gained and lost when Kunlun is drawn as a mountain, axis, center, or floating world?
- Which modern “global map” comparisons are sourced hypotheses, and which are contemporary worldbuilding?

Use these questions to prevent an impressive map from becoming false precision.
